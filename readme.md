# MessengerApp

Небольшой фронтенд-проект по шаблонизации в рамках Sprint_1 на Vite + TypeScript.
Vite использует плагин [`vitePluginMtmp`](vite-plugin-mtmp.ts) парсит файлы '.mtmp' с html версткой в типизированную фукцию, возращающую строку, заменяя значения в {{}} на аргументы функции.

## Быстрый старт

Установите зависимости и запустите дев-сервер:

```bash
npm install
npm run dev
```

Сборка production:

```bash
npm run build

```

Запуск production:

```bash
npm run start

```
